Transform models
User information

Convert NIEM model descriptions into supported formats, including NIEM JSON.

Steps:

  • Choose one or both of the following:
    • Select input format and output formats for a model transformation
    • Select input format and output formats for sample instance transformations
  • Upload files or select option to use demo files (no upload required)
  • Select transformation options

Notes:

  • Note: Hold down the Ctrl or Cmd key to select multiple or de-select options
Developer information
  • The lists of options in the four select boxes for available input and output formats are pulled from the NIEM API rather than hardcoded. These options are cached locally by the app and only updated once a new API version is available.
  • The uploaded file(s) and selected options are sent to the NIEM API for transformation. If no files are provided, the API will return samples based on the Crash Driver IEPD developed for NIEM training.
Preferences

Input

Select the input format and file for the transformation.

Output

Select desired output formats and run the transformation.

Do not upload sensitive or distribution-restricted files.
